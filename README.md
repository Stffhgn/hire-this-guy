# Hire This Guy Chrome Extension

## Overview
The **Hire This Guy** Chrome extension is a dynamic and interactive showcase of my skills and projects as a Software Developer.  
It serves as a portfolio-in-a-popup, highlighting key projects in a visually appealing and engaging format.  
The goal? To encourage potential employers to **"Hire This Guy."**  

---

## Features
- **Project Carousel:**  
  Browse through curated project highlights with seamless navigation.  
- **Built with React:**  
  Modern web technologies power the extension, demonstrating proficiency in frontend development.  
- **Custom UI/UX Design:**  
  A sleek, responsive layout styled with CSS to ensure compatibility across devices.  
- **Interactive & Engaging:**  
  Each project slide offers insight into the work, with hover effects and dynamic transitions.  

---

## How to Install and Use
### Installation
1. **Download and Unzip** the project.  
2. Open **Google Chrome** and navigate to `chrome://extensions/`.  
3. Enable **Developer Mode** (toggle in the top-right corner).  
4. Click **Load Unpacked** and select the dist folder (where `manifest.json` is located).  

### Usage
1. Click the extension icon in the Chrome toolbar.  
2. A popup will display with project details and navigation arrows.  
3. Explore the carousel to view different projects.
